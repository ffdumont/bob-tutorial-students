## 🎯 Quick Start

1. **Understand your budget**: 40 BobCoins ≈ 8M tokens total
2. **Read the main guide**: Start with [bob-best-practices-en.md](bob-best-practices-en.md)
3. **Check your project**: Review the relevant project guide
4. **Apply best practices**: Use Context Packs and workflows to optimize token usage

## 📊 Key Concepts

- **OBPAVE Loop**: Orient → Bound → Plan → Act → Verify → Explain
- **Context is a Budget**: 200K tokens per conversation
- **Contracts Before Code**: Define success criteria before execution
- **Permission Ladder**: Climb gradually from read-only to execution
